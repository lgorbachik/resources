## Eval Tools

### Frameworks & Libraries (Open Source)

| Name | Description | Link |
|------|-------------|------|
| DeepEval | pytest-native eval framework, 50+ research-backed metrics, covers RAG / agents / chatbots / safety, CI/CD ready | [github](https://github.com/confident-ai/deepeval) |
| RAGAS | Focused RAG evaluation — context relevance, groundedness, answer relevance | [github](https://github.com/explodinggradients/ragas) |
| TruLens | OpenTelemetry tracing + LLM-as-a-Judge feedback, pioneered the RAG Triad | [github](https://github.com/truera/trulens) |
| LM Evaluation Harness | EleutherAI's unified benchmark harness — standard for model comparisons | [github](https://github.com/EleutherAI/lm-evaluation-harness) |
| OpenAI Evals | Framework + open-source registry of benchmarks for evaluating LLMs | [github](https://github.com/openai/evals) |
| PromptFoo | CLI/library for prompt testing, red teaming & vulnerability scanning; used by OpenAI & Anthropic | [github](https://github.com/promptfoo/promptfoo) |
| Giskard | LLM testing & vulnerability scanning framework | [github](https://github.com/Giskard-AI/giskard) |
| Inspect AI | UK AISI's agent evaluation framework for safety & capability evals | [github](https://github.com/UKGovernmentBEIS/inspect_ai) |
